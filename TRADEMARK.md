# Ongrid Trademark and Brand Usage

The Ongrid name, logo, domain names, visual identity, and related brand assets
are brand assets of the Ongrid project. They are not licensed under the
AGPLv3 license.

The AGPLv3 license grants rights to use, copy, modify, and distribute the
source code under its terms. It does not grant permission to use Ongrid brand
assets in a way that suggests official status, sponsorship, endorsement, or
affiliation with the Ongrid project.

## Allowed uses

You may use the Ongrid name when the use is truthful and not confusing, for
example:

- referring to the upstream Ongrid open source project;
- stating that your project is based on, compatible with, or integrates with
  Ongrid;
- linking to the official Ongrid repository, website, documentation, or
  releases;
- using the Ongrid name in issue reports, tutorials, blog posts, talks, or
  documentation about the project.

## Restricted uses

You may not use Ongrid brand assets in a way that could confuse users about who
operates, maintains, sponsors, or endorses a product or service. For example,
you may not:

- present a fork, distribution, package, hosted service, SaaS offering, or
  commercial product as the official Ongrid project;
- use names such as "Ongrid Cloud", "Official Ongrid", "Ongrid Enterprise", or
  similar names for an unaffiliated service or product;
- use the Ongrid logo or visual identity as the primary branding for an
  unaffiliated product, service, or company;
- register domains, social accounts, app names, package names, or organization
  names that imply official Ongrid status or create user confusion;
- remove or alter Ongrid branding in a way that misrepresents the origin of the
  upstream project.

## Forks and distributions

You may fork and redistribute the source code under AGPLv3, but forks and
redistributions must make their independent status clear. If you materially
modify the project, use a distinct product or service name and avoid implying
that your version is maintained, endorsed, or operated by the Ongrid project.

## Commercial services

The source code license does not grant permission to use Ongrid branding for a
commercial hosted service, SaaS offering, paid distribution, support package, or
enterprise product in a way that implies official status or endorsement.

If you want to use Ongrid brand assets for a commercial service or partnership,
contact the maintainers for permission.
